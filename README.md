## Izvršni fajlovi

Izvršni fajlovi se nalaze u `'Organizator Proslava/bin/Debug/'` direktorijumu.

## Pokretanje projekta

Projekat se pokreće otvaranjem `'Organizator Proslava.exe'` koji se nalazi u direktorijumu sa izvršnim fajlovima.
